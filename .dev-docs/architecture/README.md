## 📝 O que Colocar Aqui

### ✅ Incluir

- Arquitetura geral do sistema
- Diagramas de componentes
- Decisões de design (ADRs - Architecture Decision Records)
- Padrões de código adotados
- Estrutura de módulos
- Fluxo de dados

### ❌ Não Incluir

- Guias de uso (vão em `guides/`)
- Débitos técnicos (vão em `technical-debt/`)
- Procedimentos operacionais (vão em `docs/` - feature da app)
