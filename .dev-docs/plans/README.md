## 📝 O que Colocar Aqui

### ✅ Incluir

- Roadmaps de funcionalidades
- Planos de implementação
- Estratégias de refatoração
- Cronogramas de desenvolvimento
- Propostas de melhorias grandes

### ❌ Não Incluir

- Débitos técnicos pequenos (vão em `technical-debt/`)
- Arquitetura atual (vai em `architecture/`)
- Guias de uso (vão em `guides/`)
